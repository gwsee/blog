## 关于技术
golang+vue3为主

    go主要是用的b站开源的kratos框架；
        采用了bff层和grpc通信的模式,分开了所有不同类型的应用.
      另外还涉及了mysql,ent,ali-oss

    vue3 使用的vite构建的 可以在web/pc目录下看到；大部分样式是使用v0帮我优化调整的 
      前期采用了部分的ant-design-vue的ui 后期使用的是 tailwindcss样式表
